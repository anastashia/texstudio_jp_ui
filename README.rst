=================================
 TeXstudio UI 日本語訳用ファイル
=================================

.. contents:: 目次

概要
====

TeXstudio UI 日本語訳用ファイルは、
TeXstudioのユーザーインターフェースを日本語化するためのファイルです。

利用法
======

#. ファイルtexstudio_ja.tsを、TeXstudioのシステムディレクトリの
   texstudio_*.tsが存在するディレクトリに移動

#. TeXstudioの設定で言語を日本語に変更
